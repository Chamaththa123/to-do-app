# to-do-app

## Tech Stack

### **Backend**

- Node.js
- Express.js
- MySQL
- Jest (for testing)

### **Frontend**

- React (Vite)
- Axios
- React Testing Library (for unit testing)

## Backend Setup

cd backend <br/>
npm install<br/>
npm start (for run the project)<br/>
npm test (for unit testing)<br/>
npm run test:integration (for integration testing)<br/>

## Frontend Setup

cd frontend<br/>
npm install<br/>
npm run dev (for run the project)<br/>
npm test (for unit testing)<br/>
